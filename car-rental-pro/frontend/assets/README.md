# Assets Directory

This directory contains static assets for the Car Rental Pro website.

## Structure

```
assets/
├── logo.svg          # Main logo
├── favicon.ico       # Browser favicon
└── images/           # Static images
    ├── hero-car.png  # Hero section car image
    └── brands/       # Car brand logos
```

## Usage

Place your static images here. For car photos uploaded by admin, they are stored in Cloudflare R2.
